# 🛡️ AI-Based URL Safety Checker

An intelligent cybersecurity web application that analyzes URLs and helps users identify potentially dangerous websites before visiting them.

The system combines **Machine Learning**, **Large Language Models (LLMs)**, **LangChain**, and **Groq API** to perform real-time URL risk assessment and generate AI-powered website insights.

Users can securely log in, analyze URLs, view risk scores, and receive an AI-generated summary explaining the safety and credibility of a website.

---

## 🚀 Features

### 🔐 User Authentication

* Secure Login & Signup System
* Personalized user experience
* Protected access to analysis dashboard

### 🤖 AI-Powered URL Analysis

* Real-time URL safety evaluation
* Detection of phishing and malicious websites
* AI-driven reasoning using LangChain and Groq

### 📊 Machine Learning Risk Prediction

* Predicts how risky a website is
* Generates risk scores and threat levels
* Classifies URLs as:

  * Safe
  * Suspicious
  * Malicious

### 📝 AI Website Summary

* Generates an intelligent summary of the website
* Explains potential threats and risks
* Helps users make informed decisions before visiting a link

### 📈 Analysis History

* Track previously analyzed URLs
* View past risk assessments and results

---

## ⚙️ How It Works

1. User logs into the system.
2. User enters a website URL.
3. URL features are extracted and processed.
4. Machine Learning model predicts the website's risk level.
5. LangChain interacts with Groq AI for deeper analysis.
6. The system generates:

   * Risk Score
   * Threat Classification
   * AI-Generated Website Summary
7. Results are displayed through an intuitive dashboard.
